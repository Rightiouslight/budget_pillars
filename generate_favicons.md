# Generate Favicons from App Icon

## Quick Method (PowerShell)

```powershell
# Copy existing icons as favicons
Copy-Item web\icons\Icon-192.png web\favicon.png -Force
Copy-Item web\icons\Icon-192.png web\icons\favicon-192.png -Force
Copy-Item web\icons\Icon-512.png web\icons\favicon-512.png -Force
```

## Better Method - Use Online Favicon Generator

1. **Go to:** https://realfavicongenerator.net/

2. **Upload your icon:**

   - Use `assets/icon/app_icon.png` (1024x1024)

3. **Configure:**

   - iOS: Use your icon with mint background
   - Android Chrome: Use your icon
   - Windows Metro: Use your icon
   - macOS Safari: Use your icon

4. **Download the package** and extract to `web/` folder

5. **Update `web/index.html`** with the generated code

## Manual Method - Multiple Sizes

If you have ImageMagick installed:

```powershell
# Generate different sizes
magick assets\icon\app_icon.png -resize 16x16 web\favicon-16x16.png
magick assets\icon\app_icon.png -resize 32x32 web\favicon-32x32.png
magick assets\icon\app_icon.png -resize 48x48 web\favicon-48x48.png
magick assets\icon\app_icon.png -resize 192x192 web\favicon-192x192.png
magick assets\icon\app_icon.png -resize 512x512 web\favicon-512x512.png

# Convert to .ico (for IE/Edge compatibility)
magick assets\icon\app_icon.png -define icon:auto-resize=16,32,48 web\favicon.ico
```

## Update index.html

Add these to your `<head>` section:

```html
<!-- Favicon -->
<link rel="icon" type="image/png" sizes="16x16" href="favicon-16x16.png" />
<link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png" />
<link rel="icon" type="image/png" sizes="192x192" href="favicon-192x192.png" />
<link rel="icon" type="image/png" sizes="512x512" href="favicon-512x512.png" />
<link rel="shortcut icon" href="favicon.ico" />

<!-- Apple Touch Icon -->
<link rel="apple-touch-icon" sizes="192x192" href="icons/Icon-192.png" />

<!-- Android/Chrome -->
<link rel="icon" sizes="192x192" href="icons/Icon-192.png" />
<link rel="icon" sizes="512x512" href="icons/Icon-512.png" />
```

## Current Quick Fix

Since you already have the icons generated by flutter_launcher_icons, just run:

```powershell
# Use the 192px icon as main favicon
Copy-Item web\icons\Icon-192.png web\favicon.png -Force

# Rebuild and deploy
flutter build web --release
firebase deploy --only hosting
```

The favicon will automatically be picked up from `web/favicon.png` which is already referenced in your `index.html`.
